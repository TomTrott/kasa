KASA
Plateforme de réservation de logements avec messagerie intégrée

KASA est une application full-stack permettant aux utilisateurs de :
**Rechercher et réserver des logements**
**Échanger avec les hôtes via un système de messagerie**
**Ajouter des logements à leurs favoris**
**Publier de nouveaux logements** (pour les hôtes)

## ** Pré-requis**
Avant de lancer le projet, assurez-vous d'avoir les outils suivants installés :
   Outil               | Version requise | Lien de téléchargement                     |
 |---------------------|-----------------|--------------------------------------------|
 | **Node.js**         | ≥ 18.x          | [nodejs.org](https://nodejs.org/)          |
 | **npm** ou **yarn** | ≥ 8.x           | Inclus avec Node.js                        |
 | **Git**             | ≥ 2.x           | [git-scm.com](https://git-scm.com/)        |
 | **MySQL**           | ≥ 8.x           | [mysql.com](https://www.mysql.com/)        |
